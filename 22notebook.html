vue项目

        elementUI使用
        下载    npm install element-ui -s
        引用    import elementUI from "element-ui"
        引入css import 'element-ui/lib/theme-chalk/index.css'
        使用中间件  Vue.use(elementUI)

        在el-input的标签中使用字体图像加入属性  prefix-icon="myicon myicon-user"

        input数据的验证
        
        Form 组件提供了表单验证的功能，只需要通过 rules 属性传入约定的验证规则，并将 Form-Item 的 prop 属性设置为需校验的字段名即可。
         
        rules: {
            username: [
              { required: true, message: '请输入用户名', trigger: 'blur' }
            ]
          }

        验证第二次数据
        this.$refs['loginForm'].validate(valid => {
            if(valid){
                //成功发送的处理函数
            }else{
                //无效发送的处理函数,最后必须有return false
                return false
            }
        })

        自动消失的通知配置
        this.$message({
            message: '登陆成功',        //通知消息  
            type: 'success'             //颜色success,info,error,warning
          })
